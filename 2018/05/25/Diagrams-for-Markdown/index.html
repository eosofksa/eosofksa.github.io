Markdown에서 다이어그램 그리기

2018-05-25
DevLog

가능하면 좋겠어서 해봤다.

시작

갑자기 뭔가 그래프든 플로우차트든 깔끔하게 할 수 있으면 좋겠다는 생각이 들어서 찾아봤다.

Flowchart

Flowchart의 경우 다행히 npm이 있어 어렵지 않게 구현할 수 있었다. 커스터마이제이션의 경우 기본 제공하는 option이 상당히 직관적으로 되어 있어 비교적 편리한 편. 단, flowstate쪽 설정을 유지할 경우 flowstate를 지정하지 않은 node의 설정이 마음대로 잘 되지 않는다. 필자의 경우 굳이 색이 다양한걸 선호하지 않기에 flowstate를 삭제하였다. 필요한 누군가의 경우 아래 참고 링크 중 두번째 링크를 참고하여 추가하면 될 듯.

참고)
https://www.npmjs.com/package/hexo-filter-flowchart
https://flowchart.js.org/

Sequence

Sequence의 경우 js-sequence를 통해 지원하나, 커스터마이제이션이 css로 이루어져있지만 참고할 예시가 없어 더 찾아본 후 적용 예정이다.

참고)
https://www.npmjs.com/package/hexo-filter-sequence

Chart / Graph

못 찾았다. 왜인지 모르겠으나 검색시 대부분 R에 관한 내용이 나오며 이에 대해 쓰인 문서 형식이 markdown이거나 한 경우가 많아 좀 더 오래 걸릴 듯함.
